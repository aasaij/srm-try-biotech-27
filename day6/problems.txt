1. DNA Sample Length Calculator

A biotechnology lab receives DNA samples from different patients. Each DNA sequence is recorded as a string.

Problem Statement:
Write a Python program that takes a DNA sequence as input and prints the total length of the DNA sequence.

Example Input

ATGCGTACGTA

Expected Output

Length of DNA sequence: 11

---

2. Count Nucleotides in DNA

A lab technician wants to know how many A, T, G, and C nucleotides are present in a DNA sample.

Problem Statement:
Write a Python program to count the number of A, T, G, and C in a DNA sequence.

Example Input

ATGCGATTA

Expected Output

A: 3
T: 3
G: 2
C: 1

---

3. Temperature Check for Bacterial Growth

In a biotechnology lab, bacteria grow best between 35°C and 38°C.

Problem Statement:
Write a Python program that takes temperature as input and prints:

"Optimal for growth" if temperature is between 35 and 38

"Not optimal for growth" otherwise.


Example Input

36

Output

Optimal for growth

---

4. Protein Sample Weight Analyzer

A lab technician measures the weight of protein samples.

Problem Statement:
Write a Python program that takes the weight of a protein sample as input.

If weight is:

less than 50 mg → "Low Sample"

between 50 and 100 mg → "Normal Sample"

greater than 100 mg → "High Sample"

---

5. Average Enzyme Activity

A scientist records enzyme activity readings from 5 experiments.

Problem Statement:

A microbiologist counts bacteria colonies in different petri dishes.

7. Even/Odd Cell Count Check

A cell counter machine gives the number of cells detected in a sample.

Problem Statement:

Input: 245

8. RNA Sequence Validation

RNA sequences should contain only A, U, G, C.

Problem Statement:
Write a Python program that checks whether a given RNA sequence is valid or not.

If any other character appears, print Invalid RNA Sequence.

9. Patient Sample ID Generator

A lab stores samples using the format:

BIO + PatientNumber

Example:

BIO101
BIO102

10. GC Content Percentage

GC content is important in genetics.

Problem Statement:
Write a Python program to calculate GC percentage in a DNA sequence.

Formula:

GC% = (Number of G + Number of C) / Total Length × 100

Example Input

ATGCGC

Output

GC Content: 66.67%

Drug Dosage Calculator

A biotech company produces a medicine where dosage depends on patient weight.

Rules:

Weight < 40 kg → 250 mg

Weight 40–60 kg → 500 mg

Weight > 60 kg → 750 mg


Problem Statement:
Write a Python program that takes patient weight and prints the correct dosage.
